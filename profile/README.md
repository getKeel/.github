We build **harnesses that make coding agents trustworthy**. Our first tool, [klyd](https://github.com/getklyd/klyd), extracts and enforces architectural decisions across every session, so your codebase never loses its spine.

### What we believe

- Agents should amplify human intent, not erode it.
- Specifications are the new source code — and they must be **living** documents.
- The terminal is the universal developer platform; klyd works with every agent, everywhere.
