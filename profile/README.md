We build **harnesses that make coding agents trustworthy**. Our first tool, [keel](https://github.com/getkeel/keel), extracts and enforces architectural decisions across every session, so your codebase never loses its spine.

### What we believe

- Agents should amplify human intent, not erode it.
- Specifications are the new source code — and they must be **living** documents.
- The terminal is the universal developer platform; keel works with every agent, everywhere.
